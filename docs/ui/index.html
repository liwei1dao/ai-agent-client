📐 v2 设计变化说明

导航结构（3 Tab）
🤖 Agents（首页面板）
🧪 服务中心（管理+演示）
⚙️ 全局设置
Agent 面板
列表展示所有 Agent 卡片
区分聊天(紫色)/翻译(蓝色)类型
显示关联服务徽章 + 最近使用时间
添加 Agent 流程
名称 → 类型 → 选已配置服务 →
音色选择（TTS选后展开）→
提示词（聊天）/ 语言对（翻译）
服务中心
统一管理所有已配置服务
+ 添加服务按钮一键打开弹窗
下方保留各能力快速测试卡片
添加服务流程
类型(STT/TTS/LLM/翻译) →
厂商选择 → API Key 填写 →
测试连接 → 保存（一次配置，反复复用）
补充配置项
提示词：在添加 Agent 时配置
TTS 音色：选择 TTS 服务后展开音色列表
全局设置：语速/音调/采样率/历史条数
