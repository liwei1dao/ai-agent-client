系统提示词（聊天）/ 语言对（翻译）
三种输入模式
⌨️ 文本：键盘输入 + 发送按钮
🎤 短语音：按住录音、松开发送（上滑取消）
📞 通话/同传：VAD 自动触发、连续对话
统一交互
聊天与翻译共用同一套输入栏 + 气泡组件
翻译 Agent 仅语言栏不同，无独立模式切换
通话模式：计时 + 静音 + 挂断 + TTS 朗读条
服务库（一次配置全局复用）
添加服务：类型→厂商→API Key→命名保存
Agent 配置中只需选服务名称引用
支持测试连接验证 API Key 可用性
